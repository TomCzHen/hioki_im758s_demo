Querying measurement data

Syntax

Command

 

Query

:MEASure?   <no data/ ALL>
(ALL can be specified only in continuous measurement mode.)

Response

<Measurement data>
(The response format varies depending on the mode.)

ASCII format

<LCR mode>


<ANALYZER mode>


Binary format

<LCR mode>


<ANALYZER mode>


Parameter


Explanation

Command

Query

Returns the status, judgment results, measurement values, and panel number. Set the response content with the :MEASure:VALid command.
You can select the type of parameters to return with the :MEASure:ITEM command.
For details, refer to the :MEASure:ITEM command.
Switching between ASCII mode and binary mode can be set with the :FORMat:DATA command.
During continuous measurement, a message terminator or slash (/) is inserted for each single panel measurement.
When :MEASure? : Message terminator
When :MEASure? ALL : Slash (/)

Example

Command

Query

:MEASure?

Response

ASCII format

<LCR mode>


<ANALYZER mode>


Binary format

<LCR mode>


<ANALYZER mode>


注記

In the query example, the following are set before sending the :MEASure? command.
:MEASure:ITEM 0,0,0
:MEASure:VALid 31

Response(MEASure? or :MEMory?)

LCR mode/ANALYZER mode/CONTINUOUS mode
